## ğŸ”§ Troubleshooting

### **Videos Not Displaying?**
- Check iframe source URL is correct
- Verify video privacy settings allow embedding
- Test with browser console open for errors
- Ensure video host allows iframe embedding

### **Animations Not Working?**
- Verify JavaScript is enabled
- Check browser supports Intersection Observer
- Clear browser cache and hard reload
- Test in different browser

### **Mobile Layout Issues?**
- Check viewport meta tag is present
- Verify media queries are loading
- Test on real device, not just browser resize
- Use Chrome DevTools device emulation

### **GitHub Pages Not Updating?**
- Allow 5-10 minutes for changes to propagate
- Check GitHub Actions tab for build status
- Verify index.html is in root directory
- Clear browser cache after deployment
